# Web Sync Metronome — 에이전트용 아키텍처 가이드

이 문서는 코드베이스를 탐색·수정할 때 참고할 **설계 전제와 한계**만 담는다. 제품 소개·사용법·구현 세부는 `README.md`와 소스를 본다.

## 설계상 전제와 한계

- **네트워크 동기화**는 “샘플 단위 동시 재생”이 아니라 **설정·재생 상태 공유** 수준이다. 기기마다 클럭·지연이 다르므로 멤버 측 **Latency Offset**으로 보정하는 모델이다.
- 룸 멤버십·호스트 권한은 서버에서 검증하지 않는다(교육/프로토타입 수준 신뢰 모델).
